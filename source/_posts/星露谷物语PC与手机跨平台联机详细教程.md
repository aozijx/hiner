## 联机原理说明
由于星露谷物语官方未支持PC与手机跨平台联机，本教程通过**虚拟局域网**（VPN）技术实现联机。双方设备将在同一虚拟网络中，模拟本地局域网联机环境。

## 联机准备清单
| 项目          | PC端要求                     | 手机端要求                 |
|---------------|-----------------------------|---------------------------|
| 游戏版本      | Steam/1.5.6+                | 安卓1.4.5.151 / iOS 1.4.8 |
| 工具          | 贝锐蒲公英客户端             | 贝锐蒲公英APP              |
| 系统要求      | Windows 10/11               | Android 8.0+/iOS 13+      |
| 账号          | 需注册贝锐蒲公英免费账号     | 同账号登录                 |
## 详细操作步骤
网络工具配置
1. 注册账号
访问贝锐蒲公英官网，注册免费账号（每日可创建3人网络）
{% link '贝锐蒲公英' ' ' 'https://pgy.oray.com/download/' %}

2. 设备安装
   - PC端：下载Windows版客户端
   - 手机端：应用商店搜索「贝锐蒲公英」

3. 创建虚拟网络
PC端登录后点击「创建网络」→ 选择「对等网络」→ 设置网络名称（如：StardewValley）

1. 设备接入
手机端使用同一账号登录，在「网络」列表中找到创建的网络并加入

## 游戏端设置
PC作为主机
1. 启动星露谷物语 → 点击「合作」→ 选择「主持」
2. 创建农场时勾选「允许合作游戏」
3. 记录贝锐蒲公英分配的虚拟IP（如：10.0.0.2）
4. 保持软件后台在线

手机端加入
1. 打开游戏 → 点击「合作」→ 选择「加入」
2. 输入主机虚拟IP → 端口保持默认24642
3. 等待连接（首次连接需30秒-1分钟）
4. 保持软件后台在线

## 联机注意事项
1. 版本一致性
安卓版需通过Google Play或第三方商店更新至最新版，PC端建议关闭Steam自动更新

2. 网络优化
   - 建议主机使用有线网络连接
   - 双方关闭防火墙/杀毒软件（Windows Defender需放行游戏）

3. 存档兼容性
手机版存档路径：/Android/data/com.chucklefish.stardewvalley/files/Saves

## 常见问题解决
| 问题现象                 | 解决方案                                                          |
|-------------------------|------------------------------------------------------------------|
| 连接超时                 | 检查蒲公英是否保持后台运行 → 双方互ping测试网络连通性                 |
| 无法发现房间             | 主机在防火墙中开放UDP端口：**5050, 5055, 24642**                    |
| 手机端闪退               | 关闭省电模式 → 确保手机剩余内存>2GB → 清除游戏缓存                   |
| 延迟过高（>200ms）       | 主机使用蒲公英「智能组网」→ 切换至最优服务器（华东/华南节点响应最佳）   |

## 高级技巧
- 语音聊天：通过蒲公英内置的「局域网聊天」功能实现游戏内语音
- 跨区联机：海外玩家可创建网络时选择「国际BGP线路」
- 多人拓展：通过SMAPI安装Multiplayer Mod支持8人联机

>提示：长时间联机建议购买蒲公英会员（9元/月）获得更稳定的服务支持
