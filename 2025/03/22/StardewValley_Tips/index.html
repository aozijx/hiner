操作问题：进入游戏后无法移动时，切换英文输入法，按 Shift⇧ 或 Ctrl + Space

¶【基础生存】

  • 体力管理

    • 当玩家的能量消耗到0或以下（游戏中不显示负值）时会“精疲力尽”，能量到-15后，玩家会晕倒，结束这一天。若在户外晕倒会损失最多 1000。
    • 如果入睡时已经精疲力尽，或者在 2:00AM 晕倒，则次日醒来仅恢复 50% 体力。
    • 使用肌肉回复药可以消除精疲力尽的状态。如果玩家已婚，通过亲吻配偶（每日首次）消除精疲力尽的状态。
    • 温泉可免费恢复体力。

  • 时间管理

    • 凌晨2点前回家，否则晕倒扣钱。
    • 提前睡觉可手动存档。
    • 雨天优先钓鱼/挖矿。
    • 每个季节仅 28 天，过季作物会枯萎。

  • 📺 电视节目

    • 天气预报：查看次日天气
    • 算命占卜：显示当日运气（影响挖矿、钓鱼等概率）
    • 离地而居（周一/周四）：学习游戏知识
    • 酱料女皇（周日播出/周三复播）：学习烹饪食谱
    • 钓鱼信息（完成潘姆特别任务 烈酒 解锁）：当前季节可捕获鱼类

  • 商店作息

    • 🐄 玛妮牧场：周一休息
    • 🛒 皮埃尔商店：周三休息；完成献祭后每周每天都不休息
    • 🔨 铁匠铺：完成献祭后每一周周五休息

¶【工具与升级】

  • 工具

    引用站外链接
    星露谷物语Wiki
    武器
    • 在采石场左上角的矿洞最深处有一把金色镰刀

  • 🎒 背包扩展
    玩家可在皮埃尔的杂货店处购买背包的升级，背包的升级只需要足够的金币，无需其他额外的条件，开始游戏后就可购买。

    • “大型背包”（红色）需要花费 2,000，可将物品栏扩大到24格
    • “豪华背包”（蓝色）需要花费 10,000，可将物品栏扩大到36格

¶【探索与资源】

可以使用锄头在农场或小镇、野外的地上挖蚯蚓，掉落物品如煤炭、种子等等

  • 矿洞技巧

    引用站外链接
    星露谷物语Wiki
    矿物
    • 矿井总共有 120 层，每5层解锁电梯，越深的层包含的矿的种类和数量也会越多。
    • 玩家可以通过探险家公会的马龙帮忙找回遗失的武器。花费和丢失物品基础价格等值的钱购买遗失的物品。
      每次因为生命值过低而昏迷以后，可以在物品找回服务中购买一个物品或者一堆同类的物品。未买回的遗失物品会一直保存在马龙那里。
    • 沙漠矿洞（骷髅洞穴）需先获得 头骨钥匙（普通矿洞120层解锁）。

  • 秘密森林

    • 每天可砍12块硬木，需钢斧解锁入口（法师塔上方）。
    • 秘密森林中会生成各种季节性采集品，以及会攻击玩家的史莱姆。击杀这里的史莱姆有可能掉落桃花心木种子。
    • 村民不会路过秘密森林，所以可以在秘密森林的任意位置安全放置生产设备。

  • 古物
    捐赠博物馆解锁奖励。

¶【农场建设】

  • 建筑

    • 筒仓
      一旦农场里建成筒仓，在草地里使用镰刀将获取干草到筒仓里（每块草地50%的几率）。每个筒仓可以容纳240份干草。
    • 祝尼魔小屋
      解锁后，祝尼魔会帮你收菜（范围有限，别建在作物中间！）。

  • 动物与产物

    • 为领养的宠物往小碗里浇水
    • 每日抚摸动物提升心情（影响产物品质）。
    • 未喂食或心情 <70 的动物不产物品。
    • 夜间关闭仓门，避免动物遭受袭击，被袭击后的动物将被游戏移除，并有心情信息说“<动物的名字>今天似乎很紧张不安，昨晚应该发生了些事情”。
    • 动物被困在畜棚/鸡舍外面第二天将会生气。
    • 建造马厩可骑乘马匹（速度+30%），后期解锁 传送塔 快速移动。
    • 用 帽子 给马匹 换装（将帽子拖到马身上），部分帽子会改变马的外观颜色。

¶【社区中心】

  • 完成对应社区中心收集包，社区中心里相应的区域将会翻新。
  • 如果所有的收集包都被完成了，整个社区中心将被修好，祝尼魔将会离开。之后，Joja超市会倒闭，每周三皮埃尔的杂货店不再闭店。每周五克林特会前往社区中心，因此铁匠铺每周五暂停营业，不过建筑本身仍然可以进入。

¶【NPC与好感】

  • 可结婚对象

    引用站外链接
    星露谷物语
    可结婚角色

  • 婚姻

    • 单身村民的好感度上限为8心。玩家需要和任意一个单身村民达到8心的好感度才能开始恋爱。
    • 要想订婚，玩家必须送一个美人鱼吊坠给求婚对象。求婚对象接受美人鱼吊坠后，可在日历上查看婚礼日期，婚礼当天标有美人鱼吊坠的图标。
    • 如果发生群体10心事件，则玩家所有的男朋友或女朋友会整整一个星期不会给玩家好脸色。
    • 将枯萎的花束送给男/女朋友便可以结束恋爱关系。收到枯萎的花束后，该村民的友谊值会降到5心，并且会有独特的反应。

  • 妒忌
    玩家结婚后，向其他单身的村民赠送其不讨厌的礼物可能会导致配偶吃醋。

    • 这种情况只会发生在接受礼物的村民已接受玩家的花束、且与配偶为相同性别时。
    • 村民是否喜欢该礼物与配偶是否妒忌无关。
    • 如果你的配偶吃醋了，你与配偶之间的好感度会降低30点，且第二天与配偶对话时，会有一条特殊的对话，提到你给别人送的礼物。

  • 提升好感

    • 💝 每天对话 + 送礼物（生日送最爱礼物+8倍好感）。
    • 💍 结婚后14心解锁专属剧情

¶【季节节日】

  • 春季复活节：买齐 草莓种子（过季无法购买）。
  • 冬季冬星节：提前准备NPC最爱的礼物（随机交换）。

¶【联机】

  • 资金管理

    • 共享
      所有联机队友共享一个钱包（注意队友偷偷花钱）
    • 独立
      每个联机队友各有一个钱包（注意队友偷你菜去卖钱）

  • 技能相关

    • 在共享资金时，当队友在线且有影响物品价格的技能时，其他玩家会享受到技能加成。

  • 多人模式福利

    • 👥 共享献祭进度
    • 🛏️ 可以上床回复体力
    • 🛠️ 分工协作：1人钓鱼1人挖矿

¶【实用工具】

  • 农场规划工具 Stardew Planner 提前设计洒水器布局和建筑位置。

¶【快捷键技巧】

  • Shift⇧ 快速购买/制作物品（5个/次，长按可批量操作）。
  • X 快速播种
  • Tab 切换物品栏
  • 鼠标滚轮快速切换物品
  • Y 打开表情轮盘
  • T 打开对话框
