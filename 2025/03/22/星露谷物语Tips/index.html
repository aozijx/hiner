¶【基础生存】

  • 体力管理

    • 当玩家的能量消耗到0或以下（游戏中不显示负值）时会“精疲力尽”，能量到-15后，玩家会晕倒，结束这一天。若在户外晕倒会损失最多 1000。
    • 如果入睡时已经精疲力尽，或者在 2:00AM 晕倒，则次日醒来仅恢复 50% 体力。
    • 使用肌肉回复药可以消除精疲力尽的状态。如果玩家已婚，通过亲吻配偶（每日首次）消除精疲力尽的状态。
    • ♨️ 温泉（矿洞解锁后，地图右上角）可免费恢复体力。

  • ⏳ 时间管理

    • ⏰ 凌晨2点前回家，否则晕倒扣钱。
    • 🌧️ 雨天优先钓鱼/挖矿
    • 每个季节仅 28 天，过季作物会枯萎
    • 提前睡觉可手动存档。

  • 📺 电视节目

    • 天气预报：查看次日天气。
    • 算命占卜：显示当日运气（影响挖矿、钓鱼等概率）。
    • 离地而居（周一/周四）：新手教程
    • 酱料女皇（周日播出/周三复播）：学习烹饪食谱。
    • 钓鱼信息（完成潘姆特别任务 烈酒 解锁）

  • 商店作息

    • 🛒 皮埃尔商店：周三休息
    • 🔨 铁匠铺：周五休息
    • 🐄 玛妮牧场：周一休息

🐝 星露谷物语蜂房使用指南 🍯

✦ 蜂房基础 ✦

蜂房可以将花朵转化为各种有价值的蜂蜜！放置蜂房后，当半径5格范围内有开放的花朵时，每4天可以收获一次蜂蜜。

制作材料需求：

×40 木材
×8 煤炭
×1 铁锭
×1 枫糖浆

解锁条件： 耕种等级3级

产出：

加工时间：

6100 分 ≈ 4 天

✦ 使用步骤 ✦

  1. 建造蜂房
  2. 将蜂房放置在花朵附近
  3. 确保花朵与蜂房间无障碍物
  4. 等待4天收获蜂蜜

✦ 花朵对应蜂蜜类型 ✦

花朵 蜂蜜类型 售价
无/野花 野蜂蜜 100
蓝爵 蓝爵蜂蜜 200
向日葵 向日葵蜂蜜 260
夏季亮片 夏季亮片蜂蜜 280
虞美人 虞美人蜂蜜 380
玫瑰仙子 玫瑰仙子蜂蜜 680

✦ 专家建议 ✦

  • 冬季无法生产蜂蜜
  • 使用避雷针保护蜂房
  • 搭配工匠职业可提升售价
  • 蜂房可以放置在温室（不会产出蜂蜜）和姜岛

×15 石头

解锁条件： 耕种等级2级

产出：

3 小时

  1. 从鸡舍收集鸡蛋
  2. 将鸡蛋放入机器
  3. 等待3小时游戏时间
  4. 收取成品蛋黄酱

×1 铜锭

视种类而定

  4. 收取加工成品

¶【NPC与好感】

  • 可结婚对象

    可结婚角色
  • 婚姻
    • 单身村民的好感度上限为8心。玩家需要和任意一个单身村民达到8心的好感度才能开始恋爱。
    • 在与某位单身村民达到8心好感后，第二天早晨皮埃尔将会寄来信件，告知玩家皮埃尔的杂货店开始出售花束（价格 200）。

“最近你似乎和镇上的某个人走得很近。如果你想示爱的话，就得送上一束漂亮的鲜花。我这儿卖的花，价钱极其公道！如果你某一天想组建一个家庭，这是第一步！”
—— 皮埃尔

  • 送出花束后，好感度上限提高到10心。NPC接受后，TA们在社交界面中将会显示“男/女朋友”字样。花束可以送给多个村民，不限性别。

将花束送给多个村民也不会有友谊值惩罚，但如果发生群体10心事件，则玩家所有的男朋友或女朋友会整整一个星期不会给玩家好脸色。

  • 将枯萎的花束送给男/女朋友便可以结束恋爱关系。收到枯萎的花束后，该村民的友谊值会降到5心，并且会有独特的反应。
  • 求婚
    赠送“美人鱼吊坠”求婚（雨天海边老水手处购买）。
    当玩家和某位可结婚村民达到10心好感之后，玩家将会收到刘易斯的一封信：

“我喜欢你而且想让你留在鹈鹕镇，所以我想给你一个建议。如果有一天你想向人求婚的话，你需要给那个人一个“美人鱼吊坠”。别担心，鹈鹕镇的居民都知道护身符的重要性。这是这片地方的古老传统。祝您好运！”
—— 刘易斯

要想订婚，玩家必须送一个美人鱼吊坠给求婚对象。求婚对象接受美人鱼吊坠后，可在日历上查看婚礼日期，婚礼当天标有美人鱼吊坠的图标。

  • 婚后生活

    • 村民与玩家结婚后，便会搬进玩家的农舍，好感度上限也从10心提高到14心。心情愉悦的配偶会在农场上帮些忙，可能会做早餐、喂食动物、修理栅栏或给农作物浇水。他们偶尔会改变你农舍的墙纸或地板，或给房间添置新的家具。与你的配偶交流时如果没有可用的对话将会触发一个拥抱。
    • 可以生育孩子。
      结婚后，将取消每周最多送两次礼物的限制，但是每天仍然只能送一个礼物。

  • 妒忌
    玩家结婚后，向其他单身的村民赠送其不讨厌的礼物可能会导致配偶吃醋。

    • 这种情况只会发生在接受礼物的村民已接受玩家的花束、且与配偶为相同性别时。
    • 村民是否喜欢该礼物与配偶是否妒忌无关。
    • 如果你的配偶吃醋了，你与配偶之间的好感度会降低30点，且第二天与配偶对话时，会有一条特殊的对话，提到你给别人送的礼物。

  • 提升好感

    • 💝 每天对话+送礼物（生日送最爱礼物+8倍好感）。
    • 💍 结婚后14心解锁专属剧情

¶【联机】

  • 资金管理

    • 共享
      所有联机队友共享一个钱包（注意队友偷偷花钱）
    • 独立
      每个联机队友各有一个钱包（注意队友偷你菜去卖钱）

  • 技能相关

    • 在共享资金时，当队友在线且有影响物品价格的技能时，其他玩家会享受到技能加成。

  • 多人模式福利

    • 👥 共享献祭进度
    • 🛏️ 可以上床回复体力
    • 💑 全员结婚需安装模组（原版限1人）
    • 🛠️ 分工协作：1人钓鱼1人挖矿

¶【隐藏技巧】

  • 垃圾桶致富：翻垃圾桶可能捡到食物、矿石（注意避开NPC视线，否则必扣好感）。
  • 祝尼魔小屋：解锁后，祝尼魔会帮你收菜（范围有限，别建在作物中间！）。
  • 马与传送
    • 建造马厩可骑乘马匹（速度+30%），后期解锁 传送塔 快速移动。
    • 🐎 用 帽子 给马匹 “换装”（将帽子拖到马身上），部分帽子会改变马的外观颜色。
