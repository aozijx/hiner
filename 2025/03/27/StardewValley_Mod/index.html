¶UI Info Suite 2

引用站外链接
UI Info Suite 2
提供有关您周围事物的有用信息，旨在帮助您了解正在发生的事情，而不会觉得自己在作弊。
引用站外链接
GitHub
对 Stardew Valley 的原始 UI Info Suite mod 进行持续维护。
当前功能包括：
  • 显示代表当前运气的图标
  • 显示经验值获取
  • 显示一个动态体验栏，该体验栏会根据您当前的工具或位置而变化
  • 显示更准确的心率水平
  • 显示有关物品鼠标悬停的更多信息，包括捆绑包仍然需要的物品
  • 随时随地查看日历和任务公告牌
  • 在需要抚摸的动物上显示图标
  • 显示作物类型和收获前天数
  • 当动物有物品产量（牛奶、羊毛）时的显示图标
  • 洒水器、稻草人、蜂巢和 junimo 小屋系列
  • 在地图上显示 NPC 位置
  • 按 Escape 键跳过介绍音
  • 当 Queen of Sauce 播出您还不知道的食谱时，显示她的图标
  • 在 Clint 升级您的某个工具时显示图标。图标将告诉您距离工具完成还有多长时间，并显示您正在升级的工具。
  • 显示可捐赠给博物馆的“完整收藏”成就项目的图标
  • 显示可为“完整货件”成就运送的物品的图标
  • 显示第二天天气状况的图标
  • …选项菜单中还添加了一个新选项卡，允许随时打开或关闭每个单独的 mod。
下载文件：
  • i18n
  • assets
  • UIInfoSuite2.csproj
  • manifest.json

¶CJB Cheats Menu

引用站外链接
CJB Cheats Menu
一个简单的游戏内作弊菜单，其中包含大量作弊。
引用站外链接
GitHub
这个 Mod 在多人游戏中有效吗？
  • 是的，所有的作弊都应该在多人游戏中有效（如果你是农场工人，除了改变时间）。这个 Mod 是本地的，所以如果只有一些玩家拥有它就很好了（无论他们是主机还是农场工人）。
  • 这些作弊会影响所有玩家：世界的变化（如更改日期或完成社区中心）、即时建筑、快速机器、用镰刀收割、增加金钱（如果分享金钱）、耐用的栅栏、自动喂食、自动宠物、无限干草、设置明天的天气和时间。这些作弊行为会影响与您位于同一位置的玩家：一击必杀和一击中断。否则，作弊通常只会影响您。
  • 其他玩家通常不会看到你在使用作弊（除非间接地，比如看到你使用速度作弊在屏幕上缩放）。但是，他们可能会安装专门检测您已安装的 Mod 的 Mod。

¶Convenient Inventory

引用站外链接
Convenient Inventory
为玩家的物品栏添加便利功能，例如快速堆叠到附近的宝箱和收藏的物品。
引用站外链接
GitHub
兼容性：
  • 支持单人游戏、分屏本地多人游戏和在线多人游戏。
  • 通过使用左肩按钮（可配置）进行收藏来支持控制器。
  • 支持用于游戏内配置编辑的通用 Mod 配置菜单。
  • 支持更大的背包和其他物品栏扩展 Mod（从 1.1.0 版本开始）。
  • 支持直接添加新物品栏槽位的 Mod（从 1.3.0 版本开始）。
  • 通过在安装时添加新的“全局”快速堆栈范围选项来支持 Chests Anywhere（从 1.5.0 版本开始）。
  • 支持佩戴更多戒指，通过修复带有最爱物品逻辑的边缘情况，因为它重命名了戒指装备槽（从 1.5.0 版本开始）。
  • 通过使用 SMAPI 内容管道来支持此 mod 加载的自定义图像资产的 Content Patcher（从版本 1.5.3 开始）。
    • 这允许内容包覆盖此 mod 加载的自定义图像资产。
下载文件：
  • i18n
  • assets
  • manifest.json
  • ConvenientInventory.csproj

¶Seasonal Outfits - Slightly Cuter Aesthetic

引用站外链接
Seasonal Outfits - Slightly Cuter Aesthetic
对所有角色精灵和肖像进行大修，以获得更可爱的美感
兼容性：
  • 这个 mod 现在与其他肖像或 sprite mod 兼容。只需将您不想使用的字符设置为 “false” 即可。肖像和 Sprite 不能单独选择。在 6.0.0 中，不再可能删除 portrait 文件夹以仅获取 sprites，因为这会破坏 mod。
  • 它与 Get Glam 和其他配置农民精灵的 mod 兼容。请务必按照下面的配置说明进行作。
  • 它还与 SVE 兼容。但是，SVE 使用 Marlon、Morris 和 Gunther 的精灵和肖像以及额外的动画。这意味着如果您使用 SVE，这些角色将看不到更可爱的版本。如果您想要一致的外观，请下载 Seasonal Outfits - Slightly Cuter Aesthetic for SVE。
